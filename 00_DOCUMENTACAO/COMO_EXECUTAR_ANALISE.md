# 🔍 COMO EXECUTAR A ANÁLISE DOS DADOS

## Opção 1: Executar o Script Python (Recomendado)

### Passo 1: Abrir o Prompt de Comando
- Pressione `Win + R`
- Digite: `cmd`
- Aperte Enter

### Passo 2: Navegar até a pasta
```bash
cd "C:\Users\User\Desktop\MAT-INE - INADIMPLENCIA - 2026"
```

### Passo 3: Executar o script
```bash
python ANALISE_DADOS.py
```

### Passo 4: Verificar resultado
- Um arquivo chamado `ANALISE_RESULTADO.txt` será criado
- Abra esse arquivo e copie o conteúdo
- Cole aqui na conversa comigo

---

## Opção 2: Se Python não estiver instalado

### Abra os arquivos no Excel:
1. Abra `VENCIDOS - 19-05-2026.xlsx`
2. Copie e cole aqui:
   - Nomes das colunas (primeira linha)
   - Primeiras 5-10 linhas de dados
3. Faça o mesmo para `NAO PAGOS - VINCENDAS - 19-05-2026.xlsx`

---

## Opção 3: Exportar como CSV

Se preferir, pode:
1. Abrir o arquivo Excel
2. Salvar como CSV (Arquivo > Salvar Como > Formato CSV)
3. Abrir o CSV em um editor de texto (Bloco de Notas)
4. Copiar o conteúdo e compartilhar comigo

---

**Qualquer uma dessas opções funcionará! 👍**
