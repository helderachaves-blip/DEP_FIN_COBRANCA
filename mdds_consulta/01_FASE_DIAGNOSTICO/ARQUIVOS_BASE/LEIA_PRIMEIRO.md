# 📥 ARQUIVOS BASE DO PROJETO

Esta pasta contém os dois arquivos que serão a base para criar todo o processo automático.

## Arquivos esperados:

### 1️⃣ `FATURAS_VENCIDAS.xlsx` (ou .csv)
- Todas as faturas que JÁ PASSARAM da data de vencimento
- Dados que virão do Synapta (exportadas conforme você faz hoje)

### 2️⃣ `FATURAS_VINCENDAS.xlsx` (ou .csv)
- Todas as faturas NÃO PAGAS que ainda NÃO VENCIRAM
- Dados do Synapta

---

## 📊 O que faremos com esses arquivos:

1. **Análise da estrutura** - Ver que colunas vocês têm
2. **Limpeza e normalização** - Preparar os dados
3. **Criação de camadas de análise:**
   - Segmentação por dias de atraso
   - Agrupamento por cliente
   - Cálculo de inadimplência
   - Priorização automática
4. **Criação do fluxo automático** - Mensagens, boletos, rastreamento
5. **Dashboard** - Visualização em tempo real
