🚀 COMO USAR A APLICAÇÃO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

3️⃣ ABRIR A APLICAÇÃO

   a) Abra Command Prompt (Windows)
   b) Cole este comando:
      cd "C:\Users\User\Desktop\MAT-INE - INADIMPLENCIA - 2026"

   c) Digite:
      python inadimplencia_app.py

   d) Aguarde abrir a janela (pode levar 5-10 segundos)


4️⃣ CLICAR EM "📥 CONSOLIDAR VENCIDAS"

   a) Clique no botão verde "📥 Consolidar Vencidas"

   b) Aguarde a consolidação (pode levar 30 segundos a 1 minuto)

   c) Você verá uma tabela aparecendo com:
      ├─ Nome do Aluno
      ├─ Qtd Títulos (quantidade de boletos vencidos)
      ├─ Valor Total (soma dos boletos)
      └─ Categoria (A ou B)


5️⃣ VISUALIZAR O RESULTADO (OPCIONAL)

   a) Clique em "📊 Visualizar Resultado"

   b) Aparecerá uma janela com estatísticas:
      ├─ Total de alunos
      ├─ Categoria A: quantos têm 1 título
      ├─ Categoria B: quantos têm >1 título
      ├─ Valor total em risco
      └─ Cobertura de telefone/email


6️⃣ GERAR RELATÓRIO PARA WHATSAPP

   a) Clique em "📄 Gerar Relatório WhatsApp"

   b) Serão criados 2 arquivos:
      ├─ relatorio_vencidas_DD-MM-YYYY_HH-MM-SS.txt
      └─ relatorio_vencidas_DD-MM-YYYY_HH-MM-SS.xlsx

   c) Ambos serão salvos em:
      C:\Users\User\Desktop\MAT-INE - INADIMPLENCIA - 2026\RELATORIOS\


7️⃣ USAR O ARQUIVO TXT PARA ENVIAR WHATSAPP

   a) Abra o arquivo .txt gerado

   b) Você verá duas seções:
      ├─ CATEGORIA A (1 título) - ENVIAR DIARIAMENTE
      └─ CATEGORIA B (>1 título) - ENVIAR 1X POR SEMANA

   c) Para cada aluno:
      ├─ Nome
      ├─ Telefone
      └─ Mensagem (PRONTA PARA COPIAR)

   d) Copie cada mensagem e cole no WhatsApp Web:
      1. Abra: https://web.whatsapp.com
      2. Clique no contato da pessoa
      3. Cole a mensagem
      4. Clique enviar


📊 ENTENDENDO AS CATEGORIAS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

CATEGORIA A: Apenas 1 título vencido
├─ Mensagem: Aviso com boleto
├─ Frequência: ENVIAR DIARIAMENTE (novos casos)
└─ Objetivo: Cobrança simples e direta

CATEGORIA B: Mais de 1 título vencido
├─ Mensagem: Abordagem de negociação
├─ Frequência: ENVIAR 1X POR SEMANA
└─ Objetivo: Retomar aluno / Parcelamento
